# To-Do List Application

A simple command-line interface (CLI) To-Do List application built using Python.

## Features
- **Add Tasks:** Save new tasks directly to a text file.
- **View Tasks:** Check your ongoing list of tasks with clean numbering.
- **Remove Tasks:** Safely delete completed tasks by entering their number.

## How to Run
1. Make sure you have Python installed.
2. Run the script using your terminal:
   ```bash
   python todolist.py
